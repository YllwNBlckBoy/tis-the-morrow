Intention Note...

I created this product and continue to create products to better myself. And, generally, I feel nothing towards possession/ownership, copies, and replicas. As long as I benefit from my work, we all can. So, I chose the Creative Commons Attribution-ShareAlike 4.0 International License because it goes with my belief that society cannot move forward if we all are not moving forward.

And with that said, I have a lot of work to do... a lot of things to write, a lot of things to build better for the sake of making people want to reuse it all.

Description

A downloadable HTML and CSS Zine.

A HTML/CSS zine: mostly a dialogue conversation between one's self reflecting on what it means to be human.

A HTML/CSS version of Human Defined. I originally created this zine first using HTML with the contents being dialogue and vector art. I converted it to a PDF; however, this is the actual HTML/CSS files packaged in a ZIP for preference. This one has some web page effects (JavaScript typing effect, sticky background on desktop, repeat background on mobile, mobile responsiveness) if extracted and opened in your browser.
